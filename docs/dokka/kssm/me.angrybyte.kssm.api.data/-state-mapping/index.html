StateMapping

data class StateMapping<S, A, D>

Keeps information about a particular state-to-state transition mapping. You should provide an action at creation time, and both start and end states surrounding the action.

The easiest way to create one is by using the DSL:

val mapping = AtoB moves A to B

where AtoB is the action, and A and B are states.

Parameters

source - the origin state (where the transition would start from)

action - the trigger/action (what makes the transition start)

destination - the goal state (where the transition would end at)

S - source state's payload type

A - action's payload type

D - destination state's payload type
