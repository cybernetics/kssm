Transition

data class Transition<S, A, D>

In most ways a structurally similar data model to StateMapping. In contrast to StateMapping, a Transition is mostly used to report already processed state changes to the outside world.

Normally you should not need to construct one yourself.

Parameters

oldState - the state from which the StateMachine transitioned

action - the action that triggered the state transition

newState - the state to which the StateMachine transitioned

S - the old state's payload type

A - the action's payload type

D - the new state's payload type
