requirePayload

fun <reified T> Action<T>.requirePayload(): T

Force casts the payload object into a non-null value of type T and returns it. If there was no payload set, this call will throw. If you use the wrong type T, this will throw.

Parameters

T - the type of payload this action carries

Exceptions

ClassCastException - when payload is of a different type from T

KotlinNullPointerException - when there's no payload at all (null case)
